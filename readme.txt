 1)Download the enrima-ws-java-client example:
   if you installed git: 
   git clone git//github.com/h-t-ren/enrima-ws-java-client.git
   if you haven't installed git:
   go to the webpage  https://github.com/h-t-ren/enrima-ws-java-client and click "zip" button to download 
 2)Install the project
   cd enrima-ws-java-client
   mvn clean install
 3)Run tomcat
  cd ui
  mvn tomcat:run
  (to stop tomcat: ctrl+c)
